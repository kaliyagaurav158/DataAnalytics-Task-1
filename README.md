# DataAnalytics-Task-1

# 📊 Super Store Sales Dashboard

## 📌 Project Overview

This project was completed as **Task 1** of the **15-Day Data Analytics & Business Intelligence Internship**. The objective was to perform **Data Cleaning, Exploratory Data Analysis (EDA), and Business Reporting** by creating an interactive **Power BI Dashboard** using the Super Store Sales dataset.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Identify key business KPIs.
- Create an interactive Power BI dashboard.
- Generate business insights through data visualization.

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **Microsoft Excel**
- **SQL**
- **Power Query**
- **DAX**

# 🧹 Data Cleaning

The following data cleaning steps were performed:

- Removed duplicate records
- Checked for missing values
- Corrected data types

- Validated numerical columns

# 📊 Business Insights

- Office Supplies generated the highest sales.
- Standard Class was the most preferred shipping mode.
- COD was the most used payment method.
- Consumer Segment contributed the highest revenue.
- Sales performance differs across regions and states.
- Interactive filters allow dynamic business analysis.
